# Authenticate Service

## Install Bun

You are install with [it's home](https://bun.sh).

## Development

1. Install library

```cmd
bun install
```

2. Running app

```cmd
bun run dev
```

## Development with docker

1. Install library

```zsh
make build
```

2. Running app

```zsh
make app
```
